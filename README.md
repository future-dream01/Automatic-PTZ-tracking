# Automatic-PTZ-tracking
 2023年电赛E题云台自动追踪程序，本项目可实现全部题目要求，主要硬件要求为**两个基本款openmv、两个二维云台、两只激光笔（红、蓝）、按键模块**，可根据自己的硬件基于本项目二次开发
# 代码结构简介：
1. `main1.py`控制红色激光校准、扫外围铅笔框、巡黑框
2. `main2.py`函数控制绿色激光跟踪红色;两功能均具备随时暂停以及再启动功能
3. `reset.py`舵机复位
4. `gongneng.py`存放所有功能函数
5. `shuju.py`接受来自按键模块的指令
6. `pid.py`负责舵机扫边框和跟踪过程中的pid运算
